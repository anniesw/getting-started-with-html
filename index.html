The Pieces of the Web - Major Concepts
1. Web browser - A program that runs on your computer to display files found on the web
2. Internet - The world's largest computer network
3. HTTP - The main protocol of the web which communicates back and forth between the internet and the servers
4. Servers - The computers that host the files that make up the web

The Pieces of HTML - Hypertext Markup Language
Text content = What you see
Markup = What it looks like (tells the web browser how to display a web page's words and images)
References to other documents like images or videos Links to other pages

HTML Tag Overview
1. There are opening and closing tags and the name of the tag is always the same
2. The opening tag (the word or letter) is surrounded by a less than and greater than symbol
3. The closing tag is exactly the same as the opening tag with the addition of a slash in front of the tag name/lettter that appears just after the less than symbol
4. The whole of the tag (with the opening and closing tag) is called an element
5. If you leave off the end tag then the tag never ends (example, if you do opening tag for italics which is and never add in an end tag of then all of the content in the document will be in italics

Some HTML Tag Basics:

Anchors
Provide the ability to create a link that when clicked on goes to a web page

Line Break
This is a void tag which allows you to put text on multiple lines. You'll notice that all of your HTML appears on a single line without a line break tag or a paragraph tag. The line break tag is an inline void tag which means it does not require an end tag. Without a line break or a paragraph break, all content in HTML appears on a single line (whitespace)

Paragraph
This tag to create a paragraph (with an invisible box arount the content which can have both height and width). It is not a void tag so it requires both an opening and a closing tag

The following HTML tags are inline elements:
Bold, Italic, Image, Href (to insert a link), Strong, Span, Break

The following are HTML tags are block elements (used to create an imaginary box around the content):
Paragraph, Form, Div (all this does is contain its contents)

HTML Document Structure
1. Start with the Document type with open and close tags that surround the entire document
2. Add a document head tag which = Meta data, Javascript (JS), CSS
3. Add a title tag (open and closed tags)
4. Add a body tag (open and closed tags) which are the contents of the document
5. Use CSS to add styles to the HTML document
